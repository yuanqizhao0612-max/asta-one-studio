# ASTA ONE Studio

ASTA ONE Studio is a static 6-image layout tool for ASTA ONE content.

## What It Does

- Upload a cover image.
- Upload up to 5 inner-page images.
- Paste finalized copy for 5 fixed sections.
- Adjust image crop position.
- Choose a footer content label.
- Export one PNG or a full ZIP package.

## Fixed Structure

1. Cover
2. 第一印象
3. 为什么想停留
4. 建议这样逛
5. 它好在哪里
6. 适合谁来

## GitHub Pages

This folder can be deployed directly with GitHub Pages.

No backend, API key, or server is required.

## Privacy

All uploaded photos and draft content stay in the current browser. They are not uploaded to a server by this app.
