# ASTA ONE Studio GitHub Pages Deployment

Upload the files in this folder to a GitHub repository, then enable GitHub Pages.

## Files to Upload

- `index.html`
- `app.js`
- `styles.css`
- `README.md`
- `.nojekyll`

## GitHub Pages Settings

1. Open the repository on GitHub.
2. Go to `Settings`.
3. Open `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Set `Branch` to `main`.
6. Set folder to `/root`.
7. Save.

The final URL will look like:

```text
https://your-username.github.io/asta-one-studio/
```

## Notes

- This is a static browser app.
- No server is required.
- No API key is required.
- Uploaded photos and draft content are stored only in the current browser.
